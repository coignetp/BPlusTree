# Description
This is a C++ project that implements a BPlusTree.

# Installation
## Prerequisites
C++ version must be at least C++11.  
CMake must be at least version 3.1.

## Compilation
```
mkdir build
cd build
cmake ..
cmake --build .
```
There are 2 output executable, one for the test and one for the benchmark.
